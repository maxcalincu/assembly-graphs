## scripts

### sag2svg.py

Создает SVG изображение произвольного SAG (simple assembly graph). Граф подается на вход в виде два-слова (см. пример использования). Два-слово состоит из списка целых чисел. Оно не обязано быть нормированным. 

Изображение имеет следующие свойства:

1. Вершины графа сохраняют индексацию из два-слова.
2. Концевые вершины покрашены в красный.
3. Ребра представлены в виде кубических [кривых Безье](https://ru.wikipedia.org/wiki/Кривая_Безье).
4. Трансверсаль графа является гладкой кривой первого порядка (т. е. первая производная трансверсальных ребер одной вершины совпадают в этой вершине).
5. Для каждого ребра ``e = (u, v)`` вектор первой производной в точках ``u, v`` пропорционален $e^{i\varphi}$ для какого-то $\varphi \in \{\frac{\pi}{6}, \frac{2\pi}{6}, \frac{4\pi}{6}, \frac{5\pi}{6}, \frac{7\pi}{6}, \frac{8\pi}{6}, \frac{10\pi}{6}, \frac{11\pi}{6} \}$.

Свойство 5 было добавлено чтобы упростить реализацию.

Пример использования:

```bash
./sag2svg.py --output graph.svg --input 1,1,2,3,2,3
```
